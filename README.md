#Algoritma ve programlama dersi proje ödevi
Bu proje, algoritma ve programlama dersi kapsamında bireysel olrak tasarlanmıştır.
#Proje içeriği
C programlama dilinde konsol tabanlı bir uzay simülasyonudur.Kullanıcıdan bazı bilgiler alınarak gezegenler ile birkaç tane fizik deneyi yapılmıştır.

#Fizik deneyleri 
Serbest düşme, yukarı atış, ağırlık, potansiyel enerji, hidrostatik basınç, arşimet kaldırma kuvveti, basit sarkaç, sabit ip gerilmesi ve asansör deneyleridir.

#Kütüphaneler
stdio.h
stdlib.h
math.h

#tanımlı makrolar
Programda negatif girilen değerlerin mutlak değere çevrilmesi için ternary operatör kullanarak bir makro tanımladımm.
